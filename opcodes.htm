The NES Codex - Programming

MOS 6502 Opcodes

Loading and Storage:

LDA - Load into A LDX - Load into X LDY - Load into Y STA - Store contents of A at Address STX - Store contents of X at Address STY - Store contents of Y at Address

Manipulation:

TAX - Transfer contents of A to X TXA - Transfer contents of X to A TAY - Transfer contents of A to Y TYA - Transfer contents of Y to A TSX - Transfer Stack Pointer to X TXS - Transfer contents of X to Stack Pointer

Mathematics:

ADC - Add contents of A and number with carry SBC - Subtract contents of A and number with carry INC - Increment contents of A by 1 DEC - Decrement contents of A by 1 INX - Increment contents of X by 1 DEX - Decrement contents of X by 1 INY - Increment contents of Y by 1 DEY - Decrement contents of Y by 1

Bitwise Operations:

AND - Bitwise AND ORA - Bitwise OR EOR - Bitwise XOR (Exclusive Or) ASL - Arithmetic Shift Left LSR - Logical Shift Right ROL - Rotate Left ROR - Rotate Right

Conditional Operations:

BEQ - Branch if contents of A are equal to number BNE - Branch if contents of A are not equal to number BPL - Branch on Plus (?) BMI - Branch on Minus (?) BVS - Branch if Overflow Flag is set BVC - Branch if Overflow Flag is cleared BCS - Branch if Carry Flag is set BCC - Branch if Carry Flag is cleared

Comparison Operations:

CMP - Compare contents of A with number CPX - Compare contents of X with number CPY - Compare contents of y with number BIT - Compare contents of address with contents of A via Bitwise AND

Jumps and Subroutines:

JMP - Jump to label JSR - Jump to subroutine (paired with RTS) RTS - Return from subroutine (returns to immediately after JSR)

Flag Operations:

SEC - Set Carry Flag CLC - Clear Carry Flag SEI - Set Interrupt Flag CLI - Set Interrupt Flag SED - Set Decimal Mode (Not available on 2A03/2A07) CLD - Clear Decimal Mode CLV - Clear Overflow Flag

Stack Operations:

PHA - Push contents of A onto Stack PLA - Pull (Pop) contents of A from Stack PHP - Push Processor Status onto Stack PLP - Pull (Pop) Processor Status from Stack

Miscellaneous:

BRK - Break (Non-Maskable Interrupt) RTI - Return From Interrupt NOP - No Operation
